# Aula 03

## Etapa 1

Conforme eu colocava meu nome, o modelo gerava várias palavras diferentes, como se estivesse tentando formar uma frase a partir das letras do meu nome. A partir da divisão de tokens, ele imprimiu:

`AlefdeJesusAtaideiaUMengenhariadeincitarotimizaLLMs`

Isso mostra como o modelo tenta interpretar e organizar partes do texto com base na tokenização.

## Etapa 3

Conforme a atividade pediu, escrevi as seguintes frases:

- "O gato come... liste as 3 palavras mais prováveis com justificativa logica seja muito criativa"
- "O gato come... liste as 3 palavras mais prováveis com justificativa logica seja muito técnica"

Ao analisar o texto e tentar prever as próximas palavras com base na lógica do contexto, o modelo retornou as mesmas três palavras nas duas perguntas:

- peixe  
- ração  
- rato  

Porém, em ordens diferentes.
